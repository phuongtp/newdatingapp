What it is?

Install:
https://nuget.org/packages/dotnet-ef/
>dotnet tool install --global dotnet-ef --version 6.0.1

Commands:
1.>cd API
  >dotnet ef migrations add InitialCreate -o Data/Migrations
  >dotnet ef database update

  Drop database
  >dotnet ef database drop

  Create Db again:
  >dotnet ef database update

  Remove migration:
  >>dotnet ef migrations remove

1. An object Relation Mapper (ORM)
2. Translates our code into SQL commands that update our tables in de DB
3. DbContext, use to interacting with the DB.
   DbContext.Users.Add(new User{Id = 4, Name = 'John'}) -->
   INSERT INTO Users(id, name) VALUES(4, John)
4. Features:
   Querying
   Change Tracking
   Saving
   Concurrency
   Transactions
   Caching
   Built-in conventions
   Configurations
   Migrations
