
1. Using the Angular CLI
   November 2021: Angular 12

   a. NodeJs 16.10 or higher: Node --version

   Install Angular CLI 12
   >npm install -g @angular/cli@12
   >ng --version

   New application:
   >ng new client --strict false
   > answer route=yes
   > css as style

2. Angular Language Service:
   Extension Settings:
   Angular: Enable-strict-mode-prompt: UNCHECKED!!!

3. Angular start:
   >cd client
   >ng serve
   http://localhost:4200
